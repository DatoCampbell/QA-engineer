**Типи тестування:**

1. Functional testing:
- security;
- interoperability;
- Functional.
2. Non-functional:
- performance (Load, Stress, Volume);
- Usability;
- Configuration;
- UI/gui;
- localization;
- internationalization.
3. Changes related:
- Regression;
- Re-testing;
- Smoke;
- Sanity.
4. Ще типи тестування:
- accessibility;
- End to Enf or e2e;
- Ad-hoc;
- install/uninstall.

**Рівні тестування:**

1. Unit, module, component testing.
2. Integration testing.
3. System testing.
4. Acceptance testing.
5. Alpha testing.
6. Beta testing.
